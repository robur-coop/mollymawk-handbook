# Mollymawk Operators Guide

You can also read a [rendered version](https://robur-coop.github.io/mollymawk-handbook/).

This book use `mdbook`. You can install them via `rustup` and `cargo`:

```sh
$ rustup install stable
$ cargo install mdbook
$ export PATH=$PATH:$HOME/.cargo/bin
$ git clone https://git.robur.coop/robur/mollymawk-handbook
$ cd mollymawk-handbook
$ mdbook serve
```

You can see the book on `http://localhost:3000`
